# Discord Meeting Minutes Bot

A smart Discord bot that **records voice channel meetings**, **transcribes speech to text**, and **generates professional meeting minutes** using OpenAI Whisper and Groq LLMs. Perfect for teams, guilds, or project groups who want a full summary of their voice chats automatically.

## Commands

| Command | Description |
|---------|-------------|
| `?join` | Bot joins your current voice channel. |
| `?leave` | Bot disconnects from the voice channel. |
| `?record` | Start recording the meeting. |
| `?stop` | Stop recording and generate meeting minutes. |
